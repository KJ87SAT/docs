コマンド

スラッシュコマンドがデフォルトのボットプレフィックスです。

[ ] の記号は必須引数を示し、( ) は任意です。

コマンド 説明 使用方法
/about Botの情報を表示します
/check_server 指定されたサーバーIDにBotが所属しているか確認します /check_server [server_id]
/member_count Botが参加しているサーバーのメンバー人数の合計を取得します
security_check サーバーの安全性を詳細にチェックします
serverinfo サーバーの詳細情報を表示します
set_mode 緊急モードやロックダウンモードを設定できます /set_mode [mode]
set_spam_setting サーバーのスパム対策設定を変更します /set_spam_setting (timeout) (spam_limit)...他3
userinfo ユーザー情報を表示します /userinfo (member)
verifier 認証パネルを作成します（荒らしユーザー判断機能搭載） /verifier [role] [message]
whitelist_add ユーザーをホワイトリストに追加します /whitelist_add [user]
whitelist_remove ユーザーをホワイトリストから削除します /whitelist_remove [user]
